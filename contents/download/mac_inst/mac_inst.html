The distribution packages for McXtrace may be downloaded via http://downloads.mcxtrace.org

Installing McXtrace 1.4 and later on Mac OS X

  1. Download the mcxtrace-1.4-macOS.tar.gz archive
  2. Unpack it and open the mcxtrace-1.4 folder in Finder
  3. Drag the McXtrace-1.4.app and the McXtrace-1.4-environment.command to your /Applications folder
  4. Right-click and chose "open" on the Check-PY-dependencies.command script
  5. PLEASE follow the on-screen instructions - and repeat opening the script until after you have been offered to take ownership of /usr/local!
  6. You should further Place this configuration file in your /Applications//McXtrace-1.4.app/Contents/Resources/mcxtrace/1.4/tools/Python/mccodelib/ folder (hint: run 'open $MCXTRACE' from the terminal and you should get a finder window in /Applications//McXtrace-1.4.app/Contents/Resources/mcxtrace/1.4/)
  7. Optionally also run the Check-PL-dependencies.command script if you want to have access to the Perl tools
Once installed you may open the McXtrace-1.4.app from /Applications. (On the first run, please right-click and choose "open"). Depending on your version of Mac OS, you may have to adjust security settings to allow "Apps from anywhere".

To use your newly installed McXtrace from the terminal, either

  • Use the McXtrace-1.4-environment.command shortcut from /Applications
  • Issue the mcxtrace-1.4-environment command in a new terminal
  • Use the File->Set as default menu point in the Python mxgui

Please report any trouble in using these instructions to mcxtrace-users@mcxtrace.org.
